# Hello World
Aplicação para mostrar um pouco da minha carreira e compartilhar os meus projetos.



## CLI ⌨️

- Rodar o projeto local

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) em seu navegador.

## Tecnologias

- ReactJS
- Tailwind Css
- Vite
- Typescript
- Jest
